How To Run

Backend
	cd cad-it-backend
	
	npm install

	nodemon app.js


Frontend
	cd cad-it-frontend

	npm install

	npm start



API endpoint
	http://localhost:3000/api/ref-data => problem 1
	
	http://locahost:3000/api/sensor => problem 2


Note
	For problem 1, please run backend and front end script
